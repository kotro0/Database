# Conference Management System

Spring Boot приложение для управления конференциями

##  Функционал
- Управление событиями (Event)
- Расписание сессий (Session)
- Регистрация участников (Registration)
- Проверка пересечений сессий
- Ограничение мест

##  Технологии
- Java 21
- Spring Boot 3.1.5
- Spring Data JPA
- H2 Database
- Maven

##  Запуск
```bash
mvn spring-boot:run
